# elastic search

## Usage

```yaml
source:
  type: elasticsearch
  config:
    base_url: elastic_server
```

## Inputs

| Key | Value | Example | Description |    |
| :-- | :---- | :------ | :---------- | :- |
| `base_url` | `string` | `http://localhost:9200` | URL of the Elastic server | *required* |
