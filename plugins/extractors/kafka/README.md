# kafka

## Usage
```yaml
source:
  type: kafka
  config:
    broker: "localhost:9092"
```
## Inputs
| Key | Value | Example | Description |    |
| :-- | :---- | :------ | :---------- | :- |
| `broker` | `string` | `localhost:9092` | Kafka broker's host | *required* |

## Outputs
| Field | Sample Value |
| :---- | :---- |
| `name` | `my-topic` |
| `source` | `kafka` |
